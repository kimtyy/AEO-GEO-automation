# AEO-GEO Automation
## AI 추천엔진 — AEO/GEO 마케팅 자동화 대시보드

소상공인의 AI 검색 노출(AEO/GEO)을 
자동으로 진단하고 처방하는 마케팅 자동화 시스템

### 주요 기능
- 5개 AI(Claude, ChatGPT, Gemini, 
  Perplexity, Grok) 언급률 실시간 모니터링
- AI 가시성 점수 자동 산출
- 콘텐츠 자동 생성 (FAQ형/추천형/보도자료형)
- 경쟁사 비교 분석
- 월간 리포트 자동 생성

### 기술 스택
- Frontend: HTML/CSS/JavaScript
- AI API: Claude, ChatGPT, Gemini
- 배포: Vercel

### 파일럿
설맥 가평현리점 (경기 가평군 조종면)
